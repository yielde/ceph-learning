ConfigurationÂ¶

Each Ceph process, daemon, or utility draws its configuration from several sources on startup. Such sources can include (1) a local configuration, (2) the monitors, (3) the command line, and (4) environment variables.

Configuration options can be set globally so that they apply (1) to all daemons, (2) to all daemons or services of a particular type, or (3) to only a specific daemon, process, or client.
